# Insights Screen - Android Assignment

This project is developed as part of an Android Developer Intern assignment.

## Features
- Jetpack Compose UI
- Custom charts using Canvas:
  - Stability area chart
  - Cycle bar chart
  - Weight chart
  - Donut chart
  - Correlation heatmap

## Architecture
- Clean modular structure:
  - screens/
  - cards/
  - charts/
  - components/
  - data/

## Tech Stack
- Kotlin
- Jetpack Compose
- Canvas API
